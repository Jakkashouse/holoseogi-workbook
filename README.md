# 나의 속도로 걷는 홀로서기

수강생·독자가 빈칸을 채우며 자기 이야기를 정리하는 Streamlit 워크북 앱입니다.

3개 장(현재의 나 / 홀로서기 / 미래와 꿈)의 질문에 답하고, 작성한 내용을 텍스트 파일로 내려받을 수 있습니다.

## 로컬 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 배포

Streamlit Community Cloud(https://share.streamlit.io)에서 이 저장소를 연결해 배포합니다.
별도의 비밀키나 데이터베이스가 필요하지 않습니다.
